## Preferences

### Appearance & Behavior

- Appearance
  - Compact mode: true
  - Widescreen tool window layout: true

- Editor
  - General
    - Editor Tabs
      - Mark Modified: true
  - Code Style
    - TypeScript
      - Tabs and indents
        - Tab size: 2
        - Indent: 2
        - Continuation indent: 2

- Material Theme: Atom One Dark
  - Contrast Mode
  - Override Accent Color from theme
  - Compact Statusbar
  - Material Fonts
  - Automatically reset color scheme

## Useful Key Bindings

- Settings: `Ctrl + Alt + S`
- Search Everywhere: `Shift + Shift`
- Insert Live Template: `Ctrl + J`
- Quick Switch panel: `Ctrl + ~`
- Reformat Code: `Ctrl + Alt + L`
- Structure: `Alt + 7`
- Change Theme
