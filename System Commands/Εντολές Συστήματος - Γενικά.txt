# System Commands - General

## Hardware
### Full report
```bash
inxi -Fxzr
```
and
```bash
lshw
```
### CPU info
```bash
lscpu
```
or
```bash
cat /proc/cpuinfo
```
### GPU info
```bash
inxi -Gx
```
### PCI devices info
```bash
lspci
```
### USB devices info
```bash
lsusb
```


## Distribution
### Kernel version
```bash
uname -r
```
### Distribution name & version
```bash
lsb_release -a
```

## Time
### Timedate
```bash
date
```
### Calendar
```bash
cal
```
### Uptime
```bash
uptime
```

## User session
### Current username
```bash
whoami
```
### Last logged in user
```bash
last
```
### Last logged in users
```bash
lastlog
```
### Session info
One liner:
```bash
who
```
Detailed:
```bash
w
```

## Modules
### List all available modules
```bash
modprobe -l
```
### List loaded modules
```bash
lsmod
```
### Load a module
```bash
modprobe <module_name>
```
### Unload a module
```bash
modprobe -r <module_name>
```
or
```bash
rmmod <module_name>
```
### Parameters list
```bash
modinfo -p <module_name>
```
or
```bash
modinfo <module_name> | grep ^parm
```
### Parameter's value
```bash
sudo cat /sys/module/module_name/parameters/parameter_name
```

## Commands
### Command Location and Documentation
```bash
info <command>
```
or
```bash
man <command>
```
### Display one-line manual page descriptions
```bash
whatis <command>
```
### Locate a command
```bash
which <command>
```
### Locate the binary, source and man page of a command
```bash
whereis <command>
```
### Execute a command at a particular time
```bash
echo "ls -als /etc" | at 15.04
```

## Processes
### List
```bash
ps aux
```
### App PID
```bash
pidof <program_name>
```
i.e: `pidof firefox`
or
```bash
pgrep program_name
```
### Background
- Send to background: `CTL-Z`
- List background processes: `bg`
- Bring a process to foreground: `fg`
- List:  `jobs`

## Aliases
### Show all
```bash
alias
```
### Reveal command alias
```bash
type <alias_name>
```
### BASH only - Turn alias into actual command
Type alias (i.e. als) and then CTR-ALT-E

## Tips & Tricks
### Webserver
```bash
python -m SimpleHTTPServer 8080
```
### Random number generator
```bash
echo $(shuf -n 6 -i 1-100 | sort -n)
```
### Generate passwords in the command line
```bash
apg -m 25 -a 1 -n 1
```
Needs package `pwgen`:
```bash
pwgen -s 25 1
```
### Find big file/folders in the system
```bash
find . -size +100000k -exec du -h {} \;
```
### Show system startup time spent in processes
```bash
sudo systemd-analyze --no-pager blame | head -n 15
```
### List all system users' cronjobs
```bash
for user in $(cut -f1 -d: /etc/passwd); do echo $user; sudo crontab -u $user -l; done
```
### Edit Cinnamon menu
```bash
cinnamon-menu-editor &
```
