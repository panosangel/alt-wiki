## Useful Key Bindings

- Settings: `Ctrl + Alt + S`
- Search Everywhere: `Shift + Shift`
- Insert Live Template: `Ctrl + J`
- Quick Switch panel: `Ctrl + ~`
- Reformat Code: `Ctrl + Alt + L`
- Structure: `Alt + 7`
- Change Theme
